# Project-1-Rohan-1  
**CS 2340 Group 1**

## Prerequisites  
- Install **Python 3**, **Django**, **Node.js**, and **npm**  
- In the `client` directory, run:
    ```bash 
    npm install
    npm run build
    ```
- In the `server` directory, run:
    ```bash
    python3 manage.py migrate
    ```

## Start the App  
```bash
python3 manage.py runserver
